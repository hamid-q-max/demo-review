# Demo Review AppP

A small Python demo app intentionally containing realistic issues for AI PR review and unit test generation.

- app.py
- auth.py

## Notes
This project is intentionally imperfect so review tools can catch:
- missing validation
- unsafe assumptions
- logic bugs
- mutable default arguments
- edge cases
- missing tests
